# PushUS licence summary

**This is a plain-language summary, not legal advice.** The [LICENSE](../LICENSE) file is the legal source of truth.

## Project licence

PushUS is licensed under **AGPL-3.0-only** (not "or later").

## What AGPL allows

- Use, modify, and self-host the software
- Commercial use, including charging for hosting or support
- Forking and distributing modified versions

## What AGPL requires

If you **modify** PushUS and **host it for users over a network**, you must make your **modified source code** available under AGPL-3.0-only to those users.

## PushUS Cloud

The official hosted PushUS Cloud service charges for managed hosting, convenience, and support. The application source remains open under AGPL-3.0-only. Paying for Cloud does **not** buy closed-source code.

## No CLA

Contributions are accepted under the project licence without a Contributor Licence Agreement.
